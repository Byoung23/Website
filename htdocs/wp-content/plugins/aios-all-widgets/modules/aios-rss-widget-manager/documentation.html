    Start of Loop
  • [feed_end_loop]
    End of Loop
  • Title and Description
  • [feed_title]

  • [feed_image]

  • Permalink
  • [feed_permalink]
    Feed url/permalink

Instruction

1. You need to configure the widget based on your preference. On the HTML part, put the static html and change the required data using the list of shortcodes below.

2. You need to put the repeated HTML part inside the loop.
Sample:
[feed_start_loop]
-insert repeated HTML part here-
[feed_end_loop]

Sample HTML


<div class="blog-wrap">
[feed_start_loop]
<div class="blog-list col-md-3 col-lg-3 col-sm-6">
<h3>[feed_title]</h3>
[feed_date]
<p>[feed_description]</p>
<a href="[feed_permalink]">Read More</a>
</div>
[feed_end_loop]
</div>
