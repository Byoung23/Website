Lists of Shorcodes

  • [ihf_loopstart]
    IHF Loop Start
  • [ihf_loopend]
    IHF Loop End
  • [price]
    List Price
  • [link]
    Listing URL
  • [address1]
    House Number and Street Name
  • [address2]
    City, State and Zip
  • [bedroom]
    Bedroom Information
  • [bathroom]
    Bathroom Information
  • [imagesrc]
    Image URL

Instruction

1. Target element is the ID or Class of the html element where you want the content to append.

2. You need to configure the widget based on your desired layout. On the HTML part, put the static html and change the required data using the list of shortcodes below.

3. You need to put the repeated HTML part inside the loop.
Sample:
[ihf_loopstart]
-insert repeated HTML part here-
[ihf_loopend]

Sample HTML


<div>
<div>Widget title</div>
[ihf_loopstart]
<div>
Address: <a href="[link]">[address1] [address2]</a>
</div>
<div>Price: [price]</div>
[ihf_loopend]
</div>
