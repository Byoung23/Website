Lists of Shorcodes

  • Loop
  • [idx_loopstart]
    Start of Loop
  • [idx_loopend]
    End of Loop
  • [link]
    Listing Details URL
  • [photo_url]
    Image URL
  • [listing_ID]
    Listing ID
  • [address]
    Street number and street name
  • [city]

    State

    Status

Instruction

1. You need to configure the widget based on your preference. On the HTML part, put the static html and change the required data using the list of shortcodes below.

2. You need to put the repeated HTML part inside the loop.
Sample:
[idx_loopstart]
-insert repeated HTML part here-
[idx_loopend]

Sample HTML


<div>Widget title</div>
<ul>
[idx_loopstart]
<li>
<h2><a href="[link]">[address]</a><h2><br/>
<div><img src="[photo_url]" /></div>
<div> [price]</div>
</li>
[idx_loopend]
</ul>
